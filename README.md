# Twitter Copilot Extension

![logo](src/static/images/main_logo_enable.png)

Twitter tools for translation and AI content generation

## Setup

Install pnpm

```bash
npm install -g pnpm
```

Install the dependencies:

```bash
pnpm install
```

## Get Started

Build the app for production:

```bash
pnpm build
```
